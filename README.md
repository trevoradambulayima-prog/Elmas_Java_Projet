# Elmas Python Project

This repository now contains:

- the original Java project archive in `Elmas_Java_Projet.tar`
- the extracted Java source in `Elmas_Java_Projet/`
- a Python desktop port in `main.py`

## Run the Python app

```powershell
python main.py
```

## What the Python version includes

- login screen
- main menu
- patient dashboard
- salary calculator
- in-memory patient search
- photo selection support

The Python app is built with Tkinter so it runs with standard Python on Windows.
